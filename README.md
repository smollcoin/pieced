# Pieced

Pieced is a Chrome extension that turns images on a webpage into sliding puzzles.

## Demo Website

Open the demo site here: [docs/demo.html](docs/demo.html)

## Demo Video

Watch the demo video: https://www.youtube.com/watch?v=ToKZ0lYNZxc

## About

Pieced replaces images with interactive sliding puzzles and tracks your progress. You can choose Easy, Medium, or Hard in the extension popup, then solve puzzles directly on the page.

## How To Play

1. Open the Pieced extension popup.
2. Choose a difficulty level.
3. Browse a page with images.
4. Click an image to activate the puzzle.
5. Use the arrow keys to move the tiles until the image is solved.
